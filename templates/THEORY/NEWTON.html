Метод Ньютона

{% endblock %} {% block content_theory %}

Пусть в уравнении \( f(x) = 0 \) функция имеет непрерывную производную \( f'(x) \neq 0\, x_n\) есть некоторое приближение к корню \(\overline{x}\) рассматриваемого уравнения. В окрестности точки \(x_n\) разложим функцию \( f(x)\) в ряд Тейлора.

\(f(x) = f(x_n) + f'(x_n)(x-x_n) + \frac{f''(x_n)'}{2!}(x-x_n)^2 + ... + \frac{f^{k}(x_n)'}{k!}(x-x_n)^k \)

и ограничимся линейным по \(x\) слагаемым включительно
\(0 = f(x) + f'(x_n)(x-x_n) \)
Отсюда
\( x \approx x_n - \frac{f(x_n)}{f'(x_n)} (!)\)
и согласно идее Ньютона левую часть этого выражения будем рассма тривать как следующее, \((n+1)\) - е, приближение некоторого итерационного процесса
\(x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} (*)\)
Формула (*) представляет метод Ньютона численного решения уравнений. Другое название - метод линеаризации, поскольку функция \(f(x)\) приближённо заменена линейной (!)

Критерий обрыва отсчёта в методе Ньютона следующие. как только абсолютная погрешность n-го приближения
\(\Delta =|x_n - \overline{x}| = |\frac{f(x_n)}{f'(\xi)}| \leq \frac{f(x_n)}{m},\)
где \(m = min_{[a,b]} |f'(x)|\) станет меньше требуемой точности \(\epsilon\)
\(\frac{|f(x_n)|}{n} \lt \epsilon \)
точность достигнута, и вычисления можно прекратить, записав ответ \(\overline{x} \approx x_n\) Свойства метода Ньютона таковы:
  1. функции, участвующие в расчетах, должны быть дифференцируемыми;
  2. вычислительный процесс (*) самоисправляющийся;
  3. нахождение всех корней, сколько бы их не было, обслуживается одним и тем же вычислительным процессом (*) — в противополож- ность методу итераций, в котором, как правило, для каждого корня приходится индивидуально строить итерационный процесс;
  4. скорость сходимости итерационного процесса высока
  5. на каждом шаге вычислений требуется вычислять производную \(f'(x_n)\)
На частичное уравнение этого единственно возможного недостатка метода Ньютона направлено введение двух методов, являющихся его следствиями, - метода секущих и метода хорд. {% endblock %}
